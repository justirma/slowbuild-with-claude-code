Slow Build with Claude Code

Two AI tools for product managers. Learn them interactively through Claude Code.

Feedback Analyzer

Paste customer feedback. Get it categorized and prioritized instantly — bugs, feature requests, UX issues, praise.

Pre-Read Generator

Paste a meeting transcript. Get a pre-read for the next meeting with decisions, action items, and open questions carried forward.

Get started

You'll need Claude Code and an Anthropic API key.

  1. Clone the repo and open it in Claude Code:
    git clone https://github.com/IrmaEdite/slowbuild-with-claude-code.git && cd slowbuild-with-claude-code && claude
  2. Run /start — pick a tool and choose your learning mode.
  3. That's it. Paste your data and go.
